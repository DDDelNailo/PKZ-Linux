# Installed Packages

## base

Core userspace package group required for a minimal Arch Linux system.

* Essential system utilities
* Shell tools
* Basic Linux environment

## linux

The Linux kernel.

* Hardware communication
* Process management
* Memory management
* Running the operating system

## linux-firmware

Firmware files used by hardware devices.

* Wi-Fi adapters
* Bluetooth devices
* GPUs
* Other hardware requiring firmware

## btrfs-progs

Utilities for managing Btrfs filesystems.

* Creating Btrfs filesystems
* Checking filesystem information
* Managing subvolumes
* Performing maintenance tasks

## NetworkManager

Network management service.

* Ethernet connections
* Wi-Fi connections
* Network configuration
* DNS management

## sudo

Privilege escalation tool.

* Running commands as root
* Administrative tasks
* Delegating system permissions

## vim

Terminal text editor.

* Editing configuration files
* Writing scripts
* General text editing

## openssh

SSH client and server implementation.

* Remote terminal access
* Secure file transfers
* Remote system administration
* SSH key authentication

## man-db

Manual page database.

* Accessing documentation
* Learning about commands and configuration files

## less

Terminal pager for viewing text files.

* Viewing long outputs
* Reading log files
* Navigating through text files

## tree

Directory structure visualization tool.

* Displaying directory hierarchies
* Exploring filesystem layouts
* Inspecting project structures
* Understanding file organization

## lsof

Lists open files and the processes using them.

* Identifying which process is using a file
* Inspecting open network connections
* Troubleshooting busy devices or mount points
* Investigating process activity

## htop

Interactive process viewer.

* Monitoring CPU usage
* Monitoring memory usage
* Viewing running processes
* Managing processes interactively

## btop

Advanced system resource monitor.

* Monitoring CPU usage
* Monitoring memory usage
* Monitoring disk activity
* Monitoring network activity
* Viewing running processes

## strace

System call tracing utility.

* Debugging programs
* Understanding program behavior
* Inspecting file access
* Inspecting process interactions with the kernel

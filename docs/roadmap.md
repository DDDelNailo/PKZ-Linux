# Roadmap

## Phase 1 - Base System

Goal: Install Arch Linux manually and understand every installation step.

Requirements:

* UEFI boot
* systemd-boot
* Btrfs filesystem
* Working internet connection
* Non-root user account
* Successful reboot into installed system

Success Criteria:

* System boots without installation media.
* User can log in through a TTY.
* Network connection works.
* Basic system administration is possible.

## Phase 2 - Core Services

Goal: Create a comfortable terminal-only environment.

Topics:

* Users and groups
* sudo configuration
* Package management
* SSH
* NetworkManager
* System services
* System logging

## Phase 3 - Desktop Environment

Goal: Build a Wayland desktop from individual components.

Topics:

* Hyprland
* PipeWire
* Waybar
* Launcher
* Notifications
* Clipboard
* Portals

## Phase 4 - Applications

Goal: Build a usable daily-driver system.

Topics:

* Development tools
* Gaming
* Browsers
* Productivity software
* Backup solutions

## Phase 5 - Automation

Goal: Reproduce the entire system from scripts.

Topics:

* Package installation scripts
* Configuration deployment
* User setup
* Service configuration

## Phase 6 - Distribution

Goal: Create a custom installable ISO.

Topics:

* Archiso
* Branding
* Installer workflow
* Deployment testing
